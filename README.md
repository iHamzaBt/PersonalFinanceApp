**Elevate: Personal Finance Management App**


Key Features:

Smart Transaction Tracking: Log your Income, Expenses, and Investments with ease.

Dynamic Budgeting: Set an Overall Monthly Limit.

Set Category-Specific Limits (Food, Transport, Bills, etc.).

Smart Logic: Investments and Goals are excluded from your consumption budget to give you an accurate view of your spending.

Goal Management: Create and track savings goals with a visual progress bar.

Visual Analytics: View your balance trends over time using interactive Line Charts.

Instant Notifications: Receive alerts when you exceed your budget limits. Alerts are saved in a dedicated history log.

Security: Keep your information safe with a Fingerprint (Biometric) Lock.

----------------------------------------

Language: Java

Platform: Android Studio

Database: Room Database.

Charts: MPAndroidChart library

Material Design components with View Binding.

Animations: Lottie Android (Airbnb) - for high-quality, lightweight animations.
